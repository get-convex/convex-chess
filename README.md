# Convex

This example demonstrates how to build a fully functional chat app using React, Next.js and Convex.

The app was build within few hours. Limitation:
* The computer plays random moves.
* No optimistic updates.

## Run locally

```bash
npm install
rm -rf convex.json
npx convex init
npm run dev
```
